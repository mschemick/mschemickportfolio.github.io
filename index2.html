Cary, NC Accidents
data visualization using tableau

This project focuses on building visuals and a dashboard of the results from the Cary, NC accident project. The dashboard contains an overview of the total number of accidents, their location, how many accidents occur each month, and when accidents peak and decline during the day.

Austin Bikeshare
First project using sql

This project uses the data on the bikeshare program in Austin, Tx. The dataset contains data on bike rentals, such as the location and dropoff of each bike rental, how long they rented the bike for, what kind of subscription was used, and when they rented the bike. The goal of this project was mainly to get my feet wet as this was my first use of sql. So the focus was to perform basic queries, learning how to create tables, update tables, and alter tables.
