Projects

Austin Bikeshare
Data cleaning and EDA using sql

For this project, I used SQL to not only clean the data by checking for missing values and errors, I also enchanced the data by joining tables, and reformatting to make the data more useful when builiding a dashboard for specific goals in mind. I also perform exploratory analysis with the focus on the average trip length for each bike ride. Also I used functions in excel to obtain the day of the week and round the time to the nearest hour for each recorded trip.

Austin Bikeshare
data visualization using tableau

The objective of this dashboard was to focus on the average length of a bike ride used for the bikeshare program in Austin, Tx. The dashboard itself is interactive, which you can chose to filter the data based on kiosk, the month, day of the week, and time of day. Within the dashboard, the information shows the location of the kiosk used, the average length of trip per kiosk, and what the average length per trip was per month, day of week, and time of day.

Nashville Housing Market
Data Cleaning using sql

This project uses a dataset which contains listings of homes within the Nashville Area. The listings contain data on the details of the homes such as prices, the owner's name and address, the location of the home, and other features such as the number of bedrooms and bathrooms. The main focus was to use data cleaning techniques to resolve any errors found and optimize the dataset for better use.

Instacart
Data Exploratory Analysis using sql

The ojective of this project was to perform exploratory analysis on data provided by instacart. Overall there was six datasets which included data on orders, the departments and aisles products are located, and the products purchased. There was some data cleaning involved due to missing values that needed to be resolved, but he main focus of this project was to use sql to perform queries, subqueries, joins, and unions to explore the data.
