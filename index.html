Data Collection

In my experience, data collection is more than just finding the data and accessing it through a database. The ability to understand the goals and objectives is as important as it helps determine what type of data is needed, and whether it is accessed in a database, or research through outside means.

Data Processing

While working on an end to end data project, when it comes to each step of the project I believe the most important step is the data cleaning process. Data entry can be messy with mistakes, missing values, and anomalies. As a data analyst, I take this process very seriously, any anomaly could affect the results at the end of a project, which could steer a business in the wrong direction.

Data Visualization

As a data analyst,it is important to ensure the results of the analysis can be understood by stakeholders so they can intrepet the results and put together a plan of action. My goal is to ensure to provide the stakeholders with a story that is not only informative, but also engaging for them as well. With the ability to create fully intereactive dashboards in both Tableau and Streamlit, I can provide a fully engaging story which should provide enough information for the stakeholders into making the right decision.

Austin Bikeshare
Data cleaning and EDA using sql

This project marks my inaugural venture into utilizing SQL for data analysis. With the primary objective of honing my SQL skills, I embarked on a journey to cleanse and enhance the dataset. Employing SQL, I meticulously addressed missing values and errors, while also integrating table joins and reformatting techniques to enrich the data. The ultimate aim was to prepare the dataset for the creation of purpose-driven dashboards. Furthermore, I conducted exploratory analysis, focusing on key metrics such as the average trip length for each bike ride. To augment my SQL proficiency, I supplemented the analysis with Excel functions, demonstrating a holistic approach to data manipulation and interpretation.

Austin Bikeshare
data visualization using tableau

This interactive dashboard project centers around analyzing the average bike ride length within Austin's bikeshare program. The overarching goal is to provide insights into ride patterns and behaviors. The dashboard boasts interactivity, allowing users to filter data based on various parameters such as kiosk location, month, day of the week, and time of day. Through intuitive visualization, users can explore the average trip length per kiosk, as well as track how this average varies across different months, days of the week, and times of the day. By amalgamating location-specific data with temporal trends, the dashboard offers a comprehensive view of bike ride dynamics, aiding stakeholders in making informed decisions regarding resource allocation and service optimization within the bikeshare program.

Nashville Housing Market
Data Cleaning using sql

This data cleaning project centers on a comprehensive dataset encompassing residential listings within the Nashville Area. The dataset contains details concerning property specifics, including prices, owner identities and addresses, property locations, and distinctive features such as bedroom and bathroom counts. The primary goal of this project focuses on employing advanced data cleaning methodologies to rectify any discrepancies ensuring the data is prepared for analysis by providing accurate and reliable results.

Instacart
Data Exploratory Analysis using sql

The primary goal of this project was to perform an in-depth analysis of data sourced from Instacart using SQL. From within the data sourced by Instacart, six distinct datasets were used which contain crucial information pertaining to orders, product categorization across departments and aisles, as well as the specifics of products which were purchased. As part of the analysis, data integrity issues had to be addressed, such as missing values which prompt rigorous data cleaning procedures. However, the main focus of this project was to utilize SQL techniques, such as queries, subqueries, joins, and unions to perform a comprehensive exploratory analysis in an effort to uncover insights to be used when visualizing the data in a future project.
