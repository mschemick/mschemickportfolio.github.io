Projects

Instacart
Data Exploratory Analysis using sql

The ojective of this project was to perform exploratory analysis on data provided by instacart. Overall there was six datasets which included data on orders, the departments and aisles products are located, and the products purchased. There was some data cleaning involved due to missing values that needed to be resolved, but he main focus of this project was to use sql to perform queries, subqueries, joins, and unions to explore the data.

Nashville Housing Market
Data Cleaning using sql

This project uses a dataset which contains listings of homes within the Nashville Area. The listings contain data on the details of the homes such as prices, the owner's name and address, the location of the home, and other features such as the number of bedrooms and bathrooms. The main focus was to use data cleaning techniques to resolve any errors found and optimize the dataset for better use.

Cary, NC vehicle accidents
data cleaning and exploratory analysis using python

The objective of this project is to perform data cleaning and exploratory analysis on the dataset provided by the town of Cary, NC. The dataset provides data on car accidents that had happened around Cary, NC with other details such as the type of cars used, the location, the type of roads, and other factors such as weather. Within the dataset, there were a large number of missing values found that needed to be resolved before performing analysis on the data. The data cleaning and exploratory analysis was performed using python language.

Austin Bikeshare
data visualization using tableau

This project focuses on dashboard building techniques using data on the bikeshare program in Austin, Tx. The dashboard provides an overview of the program, such as how many bikes were rented, what day had the most bikes rented, when during the day did rentals spiked and dipped, as well as the average ride.
