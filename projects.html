Cary, NC vehicle accidents
data cleaning and exploratory analysis using python

The aim of this project is to conduct data cleansing and a comprehensive exploratory analysis on the dataset provided by the town of Cary, NC. The dataset provides records of car accidents occurring within the vicinity of Cary, NC. The records include information such as vehicle types, accident locations, road types, the time and date, and weather conditions. The dataset presented a substantial number of missing values, which by leveraging the python language, I was able to approach the issue with the use of code and data cleaning methodology. Once the cleaning process was completed, the next phase of the project goal was to extract valuable insights through exploratory analysis with the use of visualizations in preparation for building a dashboard.

Cary, NC Accidents
data visualization using tableau

This project focuses on building visuals and a dashboard of the results from the Cary, NC accident project. The dashboard contains an overview of the total number of accidents, their location, how many accidents occur each month, and when accidents peak and decline during the day.
